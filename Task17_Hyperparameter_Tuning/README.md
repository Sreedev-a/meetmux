# Task 17 - Hyperparameter Tuning

## Objective

Optimize a machine learning model using efficient hyperparameter tuning.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Optuna

## Steps Performed

1. Load dataset
2. Prepare features and target
3. Define search space
4. Perform Optuna optimization
5. Evaluate using Cross Validation
6. Save best parameters
7. Compare model performance

## Output Files

- tuning_results.csv
- best_parameters.txt
- optimization_history.png
- parameter_importance.png
- model_performance.png
