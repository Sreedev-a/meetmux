# Task 18 - Natural Language Processing

## Objective

Build an NLP pipeline for text classification using TF-IDF and Logistic Regression.

## Steps Performed

1. Load review dataset
2. Clean and tokenize text
3. Convert text using TF-IDF
4. Train Logistic Regression classifier
5. Evaluate model
6. Save trained model and vectorizer

## Output Files

- processed_reviews.csv
- model.pkl
- vectorizer.pkl
- confusion_matrix.png
- word_frequency.png
- classification_results.png
